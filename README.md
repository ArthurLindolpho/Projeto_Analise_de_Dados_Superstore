  Olá rede, gostaria de compartilhar que fiz meu segundo projeto de análise de dados com python, neste projeto peguei a base de dados da superstore, disponível em: https://raw.githubusercontent.com/andrea-caron/Repository-DA-Projects/main/Sample%20-%20Superstore.csv;
  
  Neste projeto utilizei meus conhecimentos em python para analisar alguns dados importantes, a partir desses dados, foi possível enxergar de onde sai a maior parte do lucro da empresa, que são dos produtos de tecnologia, podemos ver também os produtos que mais deram prejuízo e os que mais deram lucro, os que mais são vendidos, também foi possível enxergar até onde é 'saudável' para empresa dar descontos aos clientes, visto que a partir de uma certa porcentagem, a empresa começa a ter prejuízo. Além do mais, foi possível enxergar que o principal mês de vendas durante os anos foi novembro, logo seguido de dezembro, o que mostra que deve esxistir uma preparação maior tanto da equipe quanto do estoque para esses meses, esse aumento nas vendas durante esses meses pode ocorrer devido às festas de final de ano e outras datas comemorativas importantes. Outra informação importante que foi obtida foi sobre quais regiões davam mais lucro para a empresa, e a campeã em lucro foi a região leste.
  
  É importante ressaltar que o processo de análise de dados não é importante apenas para plotar gráficos e criar dashboards, mas sim para retirar insights e informações importantes a partir desses dados, como por exemplo, diminuir a compra dos produtos que estão dando prejuízo, entender o porque pessoas de outras regiões, como a central, estão comprando menos que pessoas de outras regiões, gerar cupons de descontos para pessoas dessa região pode ser interessante, para tentar aumentar seu engajamento na região e atrair novos compradores, outra estratégia interessante é a de fazer algumas promoções nos meses que tem menos vendas normalmente, como é o mês de fevereiro, que é o mês com o menor valor em vendas e no mês de janeiro, que é o mês que menos teve lucro outra informação importante é que o mês de dezembro, mesmo tendo menos vendas que o mês de janeiro, foi o mês que mais teve lucro.

Este gráfico mostra o Lucro obtido em cada mês, foi feita uma soma dos meses ao longo dos anos, ou seja, o mês de janeiro é representado pelos meses de janeiro de 2015 - 2018, isso mostra que existe um comportamento padrão entre os anos, ou seja, o comportamento das vendas não oscila muito, o que facilita trazer novas estratégias de venda.
